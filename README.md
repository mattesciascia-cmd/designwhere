# Designwhere — MVP Prototype

A curated platform for discovering design places, events, and experiences during Dubai Design Week — interactive mockup prototype.

Single-page, self-contained (`index.html`): real Dubai coastline map with Creek and Palm Jumeirah, a live Places directory with working filters, real photography, and an RSVP flow (before → confirmed → ticket) that opens as an overlay on whichever screen you're on.

No build step — static HTML/CSS/JS, deployed as-is.
